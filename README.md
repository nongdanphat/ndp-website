# NDP Website

Website tĩnh cho công ty NDP - Chuyên về nông nghiệp, kết nối việc làm cho chủ vườn, người nông dân và người lao động. Cung cấp mua bán phân bón, hạt giống và các sản phẩm liên quan đến nông nghiệp.

## Cấu trúc thư mục

```
ndp-website/
├── index.html                  # Trang chủ
├── header.html                 # Header component
├── styles.css                  # File CSS tùy chỉnh
├── load-header-footer.js       # Script load header/footer
├── app-scripts/                # Google Apps Script
│   ├── application-form.gs     # Script xử lý form ứng tuyển
│   └── contact-form.gs         # Script xử lý form liên hệ
├── apply/                      # Trang ứng tuyển
│   └── index.html
├── campaigns/                  # Trang chiến dịch
│   └── index.html
├── contact/                    # Trang liên hệ
│   └── index.html
├── insights/                   # Trang insights/tin tức
│   ├── index.html
│   └── *.html                  # Các bài viết chi tiết
├── join-us/                    # Trang tuyển dụng
│   ├── index.html
│   ├── full-time/              # Việc làm full-time
│   ├── internship/             # Chương trình thực tập
│   └── partners/               # Đối tác
└── assets/                     # Tài nguyên tĩnh
    ├── fonts/                  # Font Inter
    ├── icons/                  # Icon mạng xã hội
    └── images/                 # Hình ảnh
```

## Công nghệ sử dụng

- **HTML5**: Cấu trúc website
- **Tailwind CSS**: Framework CSS (sử dụng CDN)
- **JavaScript**: Xử lý menu mobile và dropdown (vanilla JS)
- **Google Apps Script**: Xử lý form liên hệ và ứng tuyển
